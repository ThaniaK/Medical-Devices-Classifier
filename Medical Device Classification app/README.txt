# Medical Device Rules File

An excel file used for recording the medical device regulation.

## Installation

Double click on Medical Device Rules.xlsx file

## Usage

```
Returns the data at "Data in object format for the application" column to fill the quizData array in smart_script.js by copy-paste this column inside the brackets.

By altering the input in other cells J column is updated and it is ready to be copied.

```
